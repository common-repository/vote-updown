Vote Up/Down
Add voting system to your single post using [show_votes] shortcode. 

== Installation ==
Go to your dashboard and add new plugin via dashboard. You can also install it manually. Simply extract zip and upload it to plugins folder.

== Frequently Asked Questions ==
Question 1. How to display vote buttons?
Answer: Use [show_votes] shortcode into single post or `<?php echo do_shortcode('[show_votes]') ?>` in templates.

Question 2. Who can vote?
Answer: Anyone can vote only but once. There is no need to login for vote.
